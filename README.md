# Updatask

## Requirements

- Node (v20)
- Pnpm

## Run Application

1. Clone repository

```
git clone https://github.com/isaul19/updatask-frontend.git
```

2. Move into directory

```
cd ./updatask-frontend
```

3. Install dependencies

```
pnpm install
```

4. Create `.env` file, copy example from `.env.template`

5. Run project

```
pnpm run dev
```
